# Genlayer Deployer v2 🚀

**Live Demo:** [Genlayer Deployer App](https://genlayer-deployer--eaglekhanx.replit.app/deploy)  
*(Hosted on Replit)*

## 📖 Overview
Genlayer Deployer v2 is a streamlined tool designed to facilitate seamless intelligent contract deployments to the Genlayer network. It abstracts the complexities of the deployment process, providing an easy-to-use endpoint to push contracts directly to Genlayer.

## 🧠 Genlayer Integration
This project integrates directly with the Genlayer protocol. By hitting the `/deploy` endpoint, the application compiles and broadcasts the transaction to the Genlayer network, utilizing its consensus mechanism for LLM-driven smart contracts.

## ✨ Features
*   **Streamlined Deployment:** Easily deploy contracts via the `/deploy` route.
*   **Replit Integrated:** Hosted natively for quick access, testing, and continuous deployment.
